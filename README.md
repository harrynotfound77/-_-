# 搅拌萃取实验：搅拌混合程度评价（计算机视觉辅助）
本项目提供一个基于 Web 的小工具，用于在搅拌萃取实验中通过图像输入（上传或摄像头拍摄）对样品的**混合程度**进行分级评价。系统面向教学演示与实验流程辅助，可作为后续引入计算机视觉指标或模型的开发基础。
## 功能特性
- 支持 上传图片 或 摄像头拍摄
- 一键评估并输出混合程度分级结果：  
  - **混合程度良好**
  - **混合程度一般**
  - **混合程度较差**
- 点击“开始测验”后显示加载过程（模拟计算耗时），提升交互体验
- 界面包含清晰的 **打分标准说明**，便于实验记录与教学使用

- Python 3.8+（建议 3.10/3.11）
- 主要依赖：
  - `gradio`
  - `opencv-python`
  - `numpy`
